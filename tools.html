health checker
RPC Health Monitor

Lightweight endpoint health checker. No external dependencies — runs anywhere Python 3.7+ is installed.

🔍 rpc_health_monitor.py

Checks latency, block height, and peer count across multiple endpoints simultaneously. Outputs JSON — easy to pipe into alerts or dashboards. Zero dependencies, single file.

# single endpoint
$python rpc_health_monitor.py http://localhost:8545
# multiple endpoints
$python rpc_health_monitor.py http://node1:8545 http://node2:8545
# with config file + interval
$python rpc_health_monitor.py --config endpoints.json --interval 30

Live RPC Checker

Check any RPC endpoint directly from your browser — no installation needed. Requires the endpoint to allow CORS from browser origins.

🔌 Endpoint Health Check
Enter one or more RPC URLs (one per line). Checks eth_blockNumber, net_peerCount, and eth_syncing.
Note: The endpoint must allow browser CORS requests. Local nodes (localhost) typically work without CORS configuration.
