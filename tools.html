health checker
RPC Health Monitor

Lightweight endpoint health checker. No external dependencies — runs anywhere Python 3.7+ is installed.

🔍 rpc_health_monitor.py

Checks latency, block height, and peer count across multiple endpoints simultaneously. Outputs JSON — easy to pipe into alerts or dashboards. Zero dependencies, single file.

# single endpoint
$python rpc_health_monitor.py http://localhost:8545
# multiple endpoints
$python rpc_health_monitor.py http://node1:8545 http://node2:8545
# with config file + interval
$python rpc_health_monitor.py --config endpoints.json --interval 30

📊 prometheus-stack

Full Prometheus + Grafana monitoring stack for QoreChain nodes. Pre-configured dashboards for block height, RPC latency, peer count, and system metrics.

$docker compose -f prometheus-stack/docker-compose.yml up -d
Prometheus :9090
Metrics collection from all RPC endpoints and system exporters
Grafana :3000
Pre-built dashboards for node health, latency, and block progression
Alertmanager :9093
Alert routing for node-down, high-latency, and sync-behind events
