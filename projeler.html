A single home for everything a QoreChain node operator needs day-to-day. Includes a Python CLI for RPC health monitoring, a Docker Compose monitoring stack with Prometheus and Grafana, incident response runbooks, and a structured upgrade checklist.

Bir QoreChain node operatörünün günlük olarak ihtiyaç duyduğu her şey tek bir çatı altında. RPC sağlık izleme için Python CLI, Prometheus ve Grafana içeren Docker Compose monitoring altyapısı, incident response runbook'ları ve yapılandırılmış bir yükseltme kontrol listesi içerir.

📡
RPC Health Monitor
Python CLI — checks endpoint health, outputs a Markdown report. Zero external dependencies.
Python CLI — uç nokta sağlığını denetler, Markdown raporu üretir. Sıfır dış bağımlılık.
📊
Monitoring Stack
Docker Compose with Prometheus and Grafana. Node metrics and alert rules included.
Docker Compose ile Prometheus ve Grafana. Node metrikleri ve alarm kuralları dahil.
🚨
Incident Runbooks
Step-by-step response guides for node-down and high resource utilization scenarios.
Node-down ve yüksek kaynak kullanımı senaryoları için adım adım müdahale kılavuzları.
⬆️
Upgrade Checklist
Pre-, during-, and post-upgrade steps with a rollback plan.
Yükseltme öncesi, sırası ve sonrası adımlar ile rollback planı.
📋
Post-Incident Report TemplateOlay Sonrası Rapor Şablonu
