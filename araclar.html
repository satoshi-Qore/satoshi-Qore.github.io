RPC Health Monitor
scripts/rpc_health_monitor.py
A Python CLI tool that reads endpoints from a JSON config, performs health checks, measures latency, and outputs a Markdown report. Zero external dependencies.
JSON yapılandırma dosyasından uç noktaları okuyan, sağlık kontrolü yapan, gecikmeyi ölçen ve Markdown raporu üretirn Python CLI aracı. Sıfır dış bağımlılık.
# Copy the example config and fill in your endpoints# Örnek yapılandırmayı kopyala
cp examples/endpoints.example.json endpoints.local.json
python scripts/rpc_health_monitor.py endpoints.local.json
Pythonstdlib onlyMarkdown output
📊
Prometheus + Grafana Stack
monitoring/docker-compose.yml
Ready-to-run Docker Compose setup: Prometheus on 9090, Grafana on 3000, both localhost-bound. Includes example Prometheus config and alert rules for high CPU, low disk, high memory, node-down.
Çalıştırmaya hazır Docker Compose kurulumu: Prometheus 9090, Grafana 3000, her ikisi de localhost'a bağlı. Örnek Prometheus yapılandırması ve yüksek CPU, düşük disk, yüksek bellek, node-down için alarm kuralları dahil.
cd monitoring
cp prometheus/prometheus.example.yml prometheus/prometheus.yml
# Replace YOUR_NODE_IP in prometheus.yml# prometheus.yml içinde YOUR_NODE_IP'yi değiştir
docker compose up -d
